# Comparação de técnicas de aprendizagem de máquina e processamento de sinal para interfaces cerebrais
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
Este projeto foi desenvolvido como um trabalho de conclusao de curso. Este Git existe para permitir a reproducao dos testes e para disponibilizar os diagramas gerados no LaTeX. 
Requisitos:
* MATLAB 2018
* [http://biosig.sourceforge.net/download.html](Biosig) para ler os arquivos GDF
* [http://www.bbci.de/competition/iv/#datasets](Dados BCI IV 2008 - dataset 2b)
* [https://github.com/ottoglass/TCC-BCI/tree/main/Framework/%2BBCISTD](Framework)

[https://github.com/ottoglass/TCC-BCI/blob/main/Tabela%20de%20Resultados.ods](Tabela de resultados)
